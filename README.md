 **StayFinder — Платформа для бронирования отелей**

 StayFinder — это современный веб-интерфейс для 
 поиска и бронирования идеального жилья. Проект 
 ориентирован на удобство пользователя (UX), 
 минималистичный дизайн и быструю навигацию по 
 лучшим предложениям в индустрии гостеприимства.

 
 Умный поиск: 
 Удобная панель фильтрации по локации, датам заезда/\
 выезда и количеству гостей прямо на главном экране.


 HTML5 & Tailwind: 
 Семантическая верстка и современные CSS-фреймворки 
 (Tailwind/Bootstrap) для стилизации.
